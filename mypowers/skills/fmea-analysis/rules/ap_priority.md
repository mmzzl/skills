# AP 措施优先级规则

AP 必须由 S/O/D 三个分数查下表得出。

## AP 处理含义

- 优先级高（H）：需要（Shall）确定适当的措施来改进失效原因（故障模式）检测机制，并提供相应的故障隔离、恢复等容错措施来避免造成失效结果
- 优先级中（M）：应当（Should）确定适当的措施来改进失效原因（故障模式）检测机制，并提供相应的故障隔离、恢复等容错措施来避免造成失效结果
- 优先级低（L）：可以（Could）确定适当的措施来改进失效原因（故障模式）检测机制，并提供相应的故障隔离、恢复等容错措施来避免造成失效结果
- 对于潜在的严重度为9-10且AP为H、M的，建议至少应由管理层评审，包括所采取的任何建议措施。这里不是对高、中、低风险的优先排序，而是对降低风险的措施的优先排序
