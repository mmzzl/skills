# D 检测度评价准则

D 越大表示越难检测；D=1 表示几乎肯定能发现失效原因，或当前已经具备检测机制。

|D|可检测性|评价准则|
|---|---|---|
|10|低|不能或不可能找出失效原因，或根本没有检测机制|
|9|低|只有很极少的机会能检测出失效原因|
|8|低|只有极少的机会能检测出失效原因|
|7|低|有很少的机会能检测出失效原因|
|6|中|有较少的机会能检测出失效原因|
|5|中|有中等的机会能检测出失效原因|
|4|高|有中上的机会能检测出失效原因|
|3|高|有较多的机会能检测出失效原因|
|2|高|有很多的机会能检测出失效原因|
|1|非常高|几乎肯定能够找出失效原因，或者目前已经具备检测机制|
