# S 严重程度评价准则

S 越大表示对客户业务影响越严重。

|S|业务影响度|后果|评价准则|
|---|---|---|---|
|10|非常高|灾难性后果|满足下列条件的一个或者多个：<br>1）造成全局性的业务中断/卡慢<br>2）大范围客户业务数据丢失<br>3）系统关键数据输出严重性错误（导致灾难性安全事故的）|
|9|非常高|灾难性后果|满足下列条件的一个或者多个：<br>1）造成大范围的业务中断/卡慢<br>2）客户部分关键数据丢失<br>3）系统关键数据输出一般性错误|
|8|高|严重性后果|满足下列条件的一个或者多个：<br>1）局部性业务中断/卡慢<br>2）客户非关键数据丢失/错误|
|7|高|严重性后果|关键功能点的全部业务中断/卡慢|
|6|中|一般性后果|关键功能点的部分业务中断/卡慢|
|5|中|一般性后果|非关键功能点的全部业务中断/卡慢|
|4|中|一般性后果|非关键功能点的部分业务中断/卡慢|
|3|低|轻微性后果|对功能没有影响，对客户使用会产生困惑或者误导|
|2|低|轻微性后果|对功能没有影响，对客户会产生一些不好的体验|
|1|极低|无明显后果|没有可见的影响|
