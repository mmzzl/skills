# 大乐透号码预测系统

基于 **MultiOutputRegressor(RandomForest)** 的号码结构映射预测模型，36 维特征 + 权重平衡训练。

> ⚠️ 大乐透每期开奖为独立随机事件，本系统仅为机器学习数据建模实验，不构成购彩建议。

---

## 目录结构

```
大乐透/
├── dlt_history.csv          # 历史开奖数据（CSV 缓存，自动下载）
├── ga_generate.py           # 遗传算法生成模拟号码池
├── data_pool/
│   └── dlt_data_pool.csv    # GA 生成的号码池（~10 万组）
└── predict/
    ├── requirements.txt     # 依赖
    ├── features.py          # 36 维特征提取
    ├── pipeline.py          # 训练 + 预测流水线
    ├── analyze.py           # 回测评估
    └── test_pipeline.py     # 单元测试 + 集成测试
```

---

## 快速开始

### 1. 安装依赖

```bash
cd 大乐透/predict
pip install -r requirements.txt
```

依赖：`pandas`, `numpy`, `scikit-learn`, `requests`

### 2. 运行完整训练 + 预测

```bash
py 大乐透/predict/pipeline.py
```

输出：训练日志 → 测试集评估指标 → 下一期预测号码

### 3. 运行回测分析

```bash
py 大乐透/predict/analyze.py
```

对后 577 期历史数据做一次性回测评估，输出每位命中率、≥2红+≥1蓝比例、MAE/RMSE、诊断对照。

### 4. 运行测试

```bash
py -m pytest 大乐透/predict/test_pipeline.py -v
```

6 个测试，覆盖特征提取、后处理、端到端流水线。

## 模型架构

### 数据构成

| 数据 | 数量 | 单条权重 | 用途 |
|------|------|----------|------|
| 历史开奖（训练） | 前 2307 期 | 38 | 训练 |
| 历史开奖（测试） | 后 577 期 | — | 评估 |
| GA 池（训练） | ~64000 条 | 1 | 训练 |
| GA 池（校验） | ~16000 条 | — | 诊断 |

历史数据占比 ≈ 58%，确保真实开奖信号主导训练。

### 36 维特征

| 分组 | 维度 | 内容 |
|------|------|------|
| 红球基础静态 | 13 | 和值、跨度、奇偶、质合、三区、连号组数、除3余数 |
| 红球扩展静态 | 6 | 均值、方差、大小号、最大连号长度、奇偶交替 |
| 双蓝静态 | 8 | 和值、跨度、奇偶、大小号、余数 |
| 时序差分 | 9 | 与上期的各项差值（GA 样本填 0）|

### 模型

```python
base_rf = RandomForestRegressor(
    n_estimators=100, max_depth=12, min_samples_leaf=5,
    max_samples=0.8, max_features=0.7, random_state=42
)
model = MultiOutputRegressor(base_rf)
```

### 后处理

取整 → 裁剪到合法区间 → 去重排序 → 按欠饱和区间补全红球

## 输入输出

### 数据格式（CSV）

```
序号,红1,红2,红3,红4,红5,蓝1,蓝2
1,10,12,17,23,29,5,10
```

### 预测输出

```
预测下期: [7, 15, 16, 21, 29, 7, 11]
```

前 5 个 = 红球 (1-35)，后 2 个 = 蓝球 (1-12)，均已排序。

## 文件说明

| 文件 | 功能 |
|------|------|
| `ga_generate.py` | 从 17500.cn 下载历史数据 → CSV 缓存 → GA 生成模拟号码池 |
| `features.py` | `extract_features(combo)` → 36 维 dict |
| `pipeline.py` | 数据加载 → 特征构建 → 2σ 过滤 → 训练 → 评估 → 预测 |
| `analyze.py` | 在全部数据上训练全量模型，对 577 期测试集输出详细回测报告 |

---

## 重要风险

1. **独立随机事件**：大乐透每期开奖独立，不存在因果预测关系
2. **GA 分布偏差**：模拟数据存在固有分布偏差，通过 2σ 过滤 + GA/真实 MAE 对比监控
3. **回归输出**：后处理补号仅做合规修正，不改变模型底层分布预测逻辑
