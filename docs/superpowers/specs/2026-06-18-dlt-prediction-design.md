# 大乐透号码预测系统设计文档

## 概述

基于方案B（结构映射）构建大乐透号码预测系统：提取号码组合的结构特征作为输入 X，号码本身作为标签 Y，训练 MultiOutputRegressor(RandomForest) 学习"特征→号码"的映射关系。

模型底层学习「号码静态统计特征→号码组合」的分布映射；同时借助时序差分特征注入期号走势信息，输入最新一期完整特征向量，推导适配当期走势的下期候选号码。

---

## 1. 数据源与权重

### 1.1 数据构成

| 数据 | 数量 | 单条权重 | 用途 |
|------|------|----------|------|
| 历史开奖（训练） | 前 2307 期 | 38 | 训练 |
| 历史开奖（测试） | 后 577 期 | — | 仅评估 |
| GA池（训练） | ~64000 条 | 1 | 训练 |
| GA池（校验） | ~16000 条 | — | 辅助诊断 |

### 1.2 权重计算

- 训练集总权重贡献：2307×38 = 87,666（历史） vs 64,000×1 = 64,000（GA）
- 历史占比 ≈ 58%，确保真实开奖信号主导训练
- `sample_weight` **仅传入训练集**，测试集/校验集不传

### 1.3 GA 池前置过滤

GA 生成后做一轮筛选再划分 64k/16k：
- 基于全部历史开奖各维度统计均值、标准差计算 2σ，超出区间直接丢弃
- 去重高度同质化号码
- 保留分布贴近真实开奖的 GA 组合，不改变总数量结构

## 2. 特征工程（36 维）

### 2.1 红球基础静态（13 维）

```
红和值, 红跨度, 红奇数, 红偶数, 红质数, 红合数,
红一区(1-12), 红二区(13-24), 红三区(25-35),
红连号组数, 红余0, 红余1, 红余2
```

### 2.2 红球扩展静态（6 维）

```
红均值(=和值/5), 红方差, 小号(1-18)数, 大号(19-35)数,
最大连号长度, 奇偶交替标记(0/1)
```

### 2.3 双蓝静态（8 维）

```
蓝和值, 蓝跨度, 蓝奇数, 蓝偶数,
蓝小号(1-6), 蓝大号(7-12),
蓝1除3余数, 蓝2除3余数
```

### 2.4 时序差分（9 维，GA 填 0）

```
和值差分, 跨度差分, 红奇数差分, 红偶数差分,
红一区差分, 红二区差分, 红三区差分,
连号差分, 红蓝总合差分
```

- 历史第 1 期无前置期，9 维差分特征统一填充 0，和 GA 样本填充规则保持一致

### 2.5 约束

- 所有样本（历史 + GA）特征维度**完全对齐**（36 维），顺序一致
- GA 样本差分特征统一填 0，不产生 NaN
- 均为整数值，无需归一化（RF 不依赖特征尺度）
- 共线性不处理（RF 对共线性不敏感）

## 4. 模型

```python
base_rf = RandomForestRegressor(
    n_estimators=100,
    max_depth=12,
    min_samples_leaf=5,
    max_samples=0.8,      # 每棵树采样 80% 样本
    max_features=0.7,     # 每棵树选 70% 特征
    random_state=42
)
model = MultiOutputRegressor(base_rf)
model.fit(X_train, Y_train, sample_weight=sample_weight_train)
```

## 5. 预测流程

```
最新一期开奖                     倒数第二期开奖
     ↓                               ↓
  当期特征计算                    上期特征(仅差分用)
     ↓                               ↓
  当期特征X_latest ←── 差分 = 当期 − 上期 ──→ 差分特征
     ↓
  合并 → X_latest(36维)
     ↓
  model.predict(X_latest) → 7 维浮点数
     ↓
  后处理:
    1. 四舍五入取整
    2. 裁剪到合法区间 (红 1-35, 蓝 1-12)
    3. 红球去重、升序排序
    4. 若不足 5 个红球，优先选取同区间、奇偶分布和当前预测红球匹配的历史高频数字，保证组合分布不极端偏离真实开奖
     ↓
  输出预测号码 [红1..红5, 蓝1, 蓝2]
```

## 6. 评估体系

### 6.1 核心业务指标（后 577 期真实测试集）

- 每个位置的精确命中率（红1~蓝2，共7个）
- 单注命中 ≥N 个的比例（N=0,1,2,3,4,5,6,7）
- 命中 ≥2 个红球 + ≥1 个蓝球的比例（对应大乐透常规最低中奖档位，作为模型实用度核心观测指标）

### 6.2 辅助数值指标

- 各位置 MAE（平均绝对误差）
- 整体 7 维 RMSE

### 6.3 诊断对照

- 对比真实测试集 vs GA 校验集的 MAE
- GA 集误差远小于真实集 → 模型偏向模拟数据 → 需提高历史权重/加强 GA 过滤

```
## 8. 重要风险说明

1. 大乐透每期开奖为独立随机事件，历史号码无因果预测关系，本系统仅为机器学习数据建模实验，无法保证提高中奖概率，不构成购彩建议。
2. GA 模拟号码为基于历史分布生成的虚拟样本，存在固有分布偏差，需定期对比 GA 校验集、真实测试集误差监控模型偏移。
3. 随机森林回归输出为连续浮点数，后处理补号、截断仅做合规修正，不改变模型底层分布预测逻辑。
```
