# Lottery Predictor Skill

**名称**: `lottery-predictor`

**描述**: 从远程文本文件获取历史大乐透（中国福利彩票）开奖数据，解析后使用简单的排除规则和频率加权，对下一期号码进行预测或给出概率分布。

**使用方法**:
```bash
# 在该目录下运行（需要 Python 3.10+）
python -m lottery_predictor
```

**功能**:
1. 下载 `https://data.17500.cn/dlt_asc.txt`（会缓存到 `~/.cache/lottery_predictor.txt`）。
2. 解析每行的前 7 个号码（红球 5 个 01‑35，蓝球 2 个 01‑12）。
3. 取最近一期作为参考，排除每个位上出现过的同一号码。
4. 对剩余候选号码统计它们在该位历史出现的频率，得到相对概率。
5. 默认输出每个位概率最高的号码作为「建议下一期」；使用 `--probabilities` 可查看完整概率分布；使用 `--history N` 可限制统计的历史期数。

**实现要点**:
- 仅使用 Python 标准库 (`urllib.request`, `csv`, `collections`).
- 处理 UTF‑8 BOM 与可能的空格。
- 将下载的文件缓存到本地防止重复下载。
- `predict_next_draw` 返回最近一期、每个位的排除集合以及每个位的候选概率映射。

**示例输出**:
```
最近一期: 22 24 29 31 35 04 11
每个位排除:
  第1位(红): [22]
  第2位(红): [24]
  第3位(红): [29]
  第4位(红): [31]
  第5位(红): [35]
  第6位(蓝): [04]
  第7位(蓝): [11]
建议下一期: 01 02 03 04 05 06 07
```

**扩展**:
- 可以替换 `predict_next_draw` 中的启发式，加入更复杂的模型（马尔可夫链、机器学习等）。
- 通过 `--probabilities` 输出完整概率表，便于进一步分析。
